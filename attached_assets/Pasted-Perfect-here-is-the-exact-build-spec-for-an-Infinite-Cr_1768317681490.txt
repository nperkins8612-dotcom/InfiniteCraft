# 5. **Combination System (No AI)**

This is the core of your game.

You need a **combination dictionary**:

```

  "water|fire": "Steam",
  "water|earth": "Mud",
  "fire|earth": "Lava",
  "air|fire": "Smoke",
  ...
}
```

### Rules:

1. **Normalize the pair**  
   Sort alphabetically:

```

```

2. **Check if the combo exists**
   - If yes ‚Üí return the result element  
   - If no ‚Üí generate a fallback name  

### Fallback Name Generator (No AI)
Use simple deterministic rules:

- If both are objects ‚Üí combine names:


- If one is abstract ‚Üí prefix:

**Important:**
